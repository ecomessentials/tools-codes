Bot Setup Guide

Introduction

This guide will help you set up and use the bot. Follow the steps below to get started.

What You Need

  • A computer with Python installed.
  • A Discord account.
  • A bot token from Discord (we'll explain how to get this).

Step 1: Get Your Bot Token

  1. Go to the Discord Developer Portal.
  2. Create a new application and add a bot to it.
  3. Copy the bot token. This is like a password for your bot.

Step 2: Set Up the Bot

  1. Download the bot files and place them in a folder on your computer.
  2. Create a file named .env in the same folder.
  3. Open the .env file and add the following lines:
    4. DISCORD_BOT_TOKEN=your_bot_token
DISCORD_CHANNEL_TOPIC_ID_TO_SEND_CODE=channel_id_for_code
DISCORD_CHANNEL_ADMIN_TOPIC_ID=channel_id_for_admin
  4. Replace your_bot_token with the token you copied earlier.
  5. Replace channel_id_for_code and channel_id_for_admin with the IDs of the Discord channels you want to use.

Step 3: Run the Bot

  1. Open a terminal or command prompt.
  2. Go to the folder where you saved the bot files.
  3. Type the following command and press Enter:
    4. python main.py

Step 4: Use the Bot

Once the bot is running, you can use it in your Discord server. Here are some commands you can try:

  • !setcookie value expires: Set a cookie for the bot to use.
    Example: !setcookie abc123 1712123456
  • !resetcookie: Clear the current cookie.
    Example: !resetcookie
  • !status: Check if the bot is working.
    Example: !status

Need Help?

If you have any questions or run into issues, contact the person who provided you with the bot files.
